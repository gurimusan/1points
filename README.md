1points homepage
================

http://1points.co.jp/

Requirements
============

- node >= 8.9
  - https://nodejs.org/ja/download/
- yarn
  - https://yarnpkg.com/lang/ja/docs/install/

Setup
=====

    $ yarn

Build
=====

単純にビルドする場合は下記のとおり。

    $ yarn run build

ファイルを監視してビルドを自動化する場合は下記のとおり。

    $ yarn run watch
